# PWA Icons

To complete the PWA setup, add the following icon files to this directory:

## Required Icons

- `icon-192.png` - 192x192 pixel PNG icon (for Android home screen)
- `icon-512.png` - 512x512 pixel PNG icon (for Android splash screen)
- `favicon.ico` - 32x32 pixel favicon (optional, replaces vite.svg)

## Icon Guidelines

- Use the Kustom Kraft Cabinets logo or branding
- Ensure good contrast for visibility on various backgrounds
- Icons should be square and centered
- PNG format with transparent background recommended

## Tools for Creating Icons

- Use online tools like [Favicon Generator](https://favicon.io/) or [PWA Icon Generator](https://tools.crawlink.com/tools/pwa-icon-generator/)
- Or create manually using design software like Figma, Photoshop, etc.

## Testing

After adding icons, test the PWA installation on:
- Android: Chrome browser > Menu > "Add to Home Screen"
- iOS: Safari > Share > "Add to Home Screen"
- Desktop: Chrome > Menu > "Install Kustom Kraft Cabinets - Job Board"
